연산자 (Operator)
====


## 증감연산자
```javascript
let i = 0;
i++; // 0
console.log(i++); // 1
```
먼저 1을 더하고 동작한다.

```javascript
let i = 0;
i++; // 0
console.log(++i); // 2
```
먼저 동작한 다음 1을 더한다.

## 대입연산자
```javascript
let i = 10
i += 2; // i = i + 2 = 12
i -= 2; // i = i - 2 = 8
i *= 2; // i = i * 2 = 20
i %= 2; // i = i % 2 = 5
```
