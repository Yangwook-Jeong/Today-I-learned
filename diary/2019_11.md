# 오늘 배운 것을 정리하는 공간

## 01.11.2019

- boto3

  - dynamodb

    - scan()

      - `FilterExpression`: 조건
      - `ProjectionExpression`: 원하는 속성
      - `ExpressionAttributeNames`: 컬럼명을 교체한다. `#<name>`을 사용한다.

        ```py
        table.scan(
          ProjectExpression = Key('id').eq(1),
          FilterExpression = 'id, #i',
          ExpressionAttributeNames = { '#i': 'info' }
        )
        ```
